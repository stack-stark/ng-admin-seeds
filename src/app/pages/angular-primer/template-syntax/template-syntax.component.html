ng-container 是 Angular 所定义的一个特殊 tag,可以在其标签内进行运算.处理,而不会影响到页面结构

*ngFor="let item of list;let idx = index;"这是循环渲染,被循环对象为list,idx则为list的index

(click)="printData(item)" 事件绑定,其中item为传递的参数

[ngStyle],[ngClass]则是内联样式和css选择器的条件绑定

*ngIf="item.id == 'a007'" *ngIf是条件渲染,条件成立则渲染,反之则销毁

item.create_time | date: 'yyyy-mm-dd hh:mm:ss' '|' 是管道符,date是angular内置管道,格式化时间
